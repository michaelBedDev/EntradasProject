# Feature: Autenticación

Esta feature maneja toda la lógica relacionada con la autenticación de usuarios, incluyendo:

- Login/Logout
- Registro de usuarios
- Gestión de sesiones
- Recuperación de contraseña

## Estructura

- `components/`: Componentes específicos de autenticación
- `hooks/`: Hooks personalizados para la gestión de autenticación
- `services/`: Servicios de autenticación (API calls, etc.)
- `actions/`: Acciones del servidor para autenticación
- `types/`: Tipos específicos de autenticación
- `lib/`: Utilidades específicas de autenticación
